# Pong-game

# 🏓 Retro Pong Arcade in Python 🕹️

Welcome to the ultimate blast from the past! Dive into the nostalgic world of classic arcade gaming with this fully functional, adrenaline-pumping, two-player Pong clone built entirely in Python using the `turtle` graphics library! 🐢✨

## ✨ Awesome Features

* 🎮 **Classic 2-Player Action:** Grab a friend and settle the score! The Left Player controls their paddle with `W` and `S`, while the Right Player dominates with the `Up` and `Down` arrow keys.
* ⚡ **Dynamic Speed Boosts:** Keep your reflexes sharp! The ball accelerates after every successful paddle hit, making long rallies incredibly intense.
* 🏆 **Live Score Tracking:** Bragging rights are displayed in real-time with a sleek, minimalist scoreboard that automatically updates when a ball gets past a paddle.
* 🎨 **Authentic Retro Aesthetic:** Complete with a meticulously drawn dashed centerline and a stark black-and-white 800x600 canvas for that true 1970s arcade feel.

## 📂 Behind the Scenes (File Breakdown)

* **`index.py`**: The beating heart of the game! It sets up the retro court, maps the keyboard controls, and runs the main collision loop.
* **`ball.py`**: Handles all the physics—from bouncing off the top and bottom walls to accelerating when smashed by a paddle.
* **`paddle.py`**: Powers the ultra-responsive paddles, allowing smooth up-and-down movement to block incoming shots.
* **`centerline.py`**: A neat little script that uses a turtle loop to draw the iconic dashed net down the middle of the screen.
* **`scoreboards.py`**: Manages the giant, nostalgic `Courier` font scores at the top of the screen.
